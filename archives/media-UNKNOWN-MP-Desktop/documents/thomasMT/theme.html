Paragraph and Copy Styles

Body Copy

This theme's global default font-size is 14px, with a line-height of 1.4px. This is applied to the <body> and all paragraphs. In addition, <p> (paragraphs) receive a bottom margin of half their computed line-height (10px by default).

<p>Nullam quis risus eget urna mollis ornare vel eu leo.</p>
--OR--
<div class="body-text">Nullam quis risus eget urna mollis ornare vel eu leo.</div>

Lead Body Copy

Make a paragraph <p> </p> or any other layout element stand out by adding a class of .lead.

<p class="lead">Vivamus sagittis lacus vel augue laoreet rutrum faucibus dolor auctor. Duis mollis, est non commodo luctus.</p>

Bold Text

For emphasizing a snippet of text with a heavier font-weight, wrap the content with a <strong><strong> tag or by using a class of "bold".

<strong>rendered as bold text</strong>
--OR--
<div class="bold">Nullam quis risus eget urna mollis ornare vel eu leo.</div> Nullam quis risus eget urna mollis ornare vel eu leo.

Building with the Grid System

By using the responsive grid system, you can quickly scaffold a responsive grid to contain your content with the built in grid classes and columns of differing widths and breakpoints. The breakpoints for the grid are:

  • small- (for small breakpoints which are smaller than 40em) -- Good for phone size and down.
  • medium- (for medium breakpoints which are from 40.063em to 64em) -- Good for tablet sizes.
  • large- (for large breakpoints which are from 64.063em and up) -- Good for desktop and above.

The grid is based on a standard 12 column layout. Start out by adding a <div></div> or other container with a class of "row" to encapsulate your columns of content. This creates a horizontal row to hold vertical columns of your content. The pattern for specifying the breakpoint class is first the breakpoint size itself (small, medium, large) followed by a dash and then the grid amount you want your column to span.

For example, if you wanted your column to span 50% of the screen at medium sizes, you would add the classes of 'medium-12 columns'. If you wanted to change it to span a third of the screen at large sizes, you would add the class of 'large-4 columns'. By default if you do not add the 'columns' class as well, the content will not clear itself up to the same line and will "float" to the next line. By adding the 'columns' class, you can create columns, side by side, totalling up to twelve.

<section class="row">
  <div class="medium-12 large-8 columns"></div>
  <div class="medium-12 large-4 columns"></div>
</section>

<section class="row">
  <div class="medium-12 large-4 columns"></div>
  <div class="medium-12 large-4 columns"></div>
  <div class="medium-12 large-4 columns"></div>
</section>

<section class="row">
  <div class="medium-6 columns"></div>
  <div class="medium-6 columns"></div>
</section>

Table Styles

Basic Table

For basic styling—light padding and only horizontal dividers—add the base class .table to any <table>. It may seem super redundant, but given the widespread use of tables for other plugins like calendars and date pickers, we've opted to isolate our custom table styles.

<table class="table">
  <thead>
    <tr>
      <th>#</th>
        <th>First Name</th>
        <th>Last Name</th>
        <th>Favorite Color</th>
        <th>Favorite Number</th>
      </tr>
    </thead>
